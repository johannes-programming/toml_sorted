Overview
--------

This project sorts TOML data.

Installation
------------

To install ``toml_sorted``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

        pip install toml_sorted

Typing
------

This project is strictly typed with ``mypy``.

Features
--------

``toml_sorted.core``
''''''''''''''''''''

``toml_sorted.core.main.main(args: Optional[Iterable[str]] = None, /) -> None``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This function is the CLI implementation of ``run``.

It accepts these instruction flags, applied in the order given:

- ``--sort`` starts a new sort instruction (ascending).
- ``--sort-reverse`` starts a new sort instruction (descending).
- ``--key=NAME`` descends into the table key ``NAME``.
- ``--index=N`` descends into the array element ``N``.
- ``--all-keys`` expands the current table or array, applying the rest
    of the instruction (any remaining ``--key``/``--index`` steps and the
    sort itself) to every value in turn.

For example::

        toml_sorted example.toml --sort --key=foo --all-keys --index=0

turns

.. code-block:: python

        {"foo": {"bar": [[4, 2], {}], "baz": [{"a": 9, "c": 8, "b": 7}]}}

into

.. code-block:: python

        {"foo": {"bar": [[2, 4], {}], "baz": [{"a": 9, "b": 7, "c": 8}]}}

``class toml_sorted.enum.Selector.Selector``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

``ALL_KEYS``
^^^^^^^^^^^^

A path token that, instead of descending into a single key or index,
expands the current table or array and applies the remaining path and
the sort to every value. It is produced by the ``--all-keys`` flag.
