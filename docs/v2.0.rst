Overview
--------

This project sorts TOML data.

Installation
------------

To install ``toml_sorted``, you can use ``pip``.
Open your terminal and run:

.. code-block:: shell

    pip install toml_sorted

Typing
------

This project is strictly typed with ``mypy``.
